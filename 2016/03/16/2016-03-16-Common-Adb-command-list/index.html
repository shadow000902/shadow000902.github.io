常用Adb命令列表

发表于 2016-03-16 更新于 2021-08-20 分类于 Android 阅读次数： Valine：
本文字数： 3k 阅读时长 ≈ 3 分钟

  1. 安装APK（如果加 -r 参数，保留已设定数据，重新安装filename.apk）

  2. 卸载APK（如果加 -k 参数，为卸载软件但是保留配置和缓存文件）

  3. 查看当前所有连接的设备id

  4. 对某一设备操作

  5. 导入文件到设备（该命令也可以实现安装APK，只要把APK导入到”/system/app”或者”/data/app”就OK）

  6. 从设备导出文件

  7. 查看log信息

  8. 进入shell模式

  9. 启动activity（可能需要root权限）

  10. 获取设备ID号

  11. 开启/关闭adb服务

  12. 访问sqlite3

  13. 重启设备

  14. 挂载分区（可使系统分区重新可写）

  15. 发布端口（可以设置任意的端口号，做为主机向模拟器或设备的请求端口）

  16. 查看最上层的Activity名字

  17. 抓取重启的log

  18. 查找占用指定端口的pid（进程id）（查找电脑被占用的端口，不是手机）

  19. monkey命令

  20. adb录屏

  21. 获取Android版本号

  22. 获取SDK版本号

  23. 获取设备型号

  24. 获取指定pid的应用包名

  25. 获取电池状态

  26. 进入bootloader

  27. 获取系统中安装的系统应用包名列表

  28. 获取系统中安装的第三方应用包名列表

  29. 模糊查询应用包名

  30. 获取启动应用所花时间

  31. 清除应用用户数据

  32. 获取设备当前显示的所有分辨率信息

  33. 查看内存限制
